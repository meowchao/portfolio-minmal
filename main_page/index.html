spikey_keeb

A handwired keyboard with an open sourse firmware QMK firmware written c++ I desigend a custom case(3d printed) it has mx blue switches with a 0.91” OLED SCREEN to display stuff like logo, gifs, etc. it has 36 keys arranged in ortholinear pattern. keycaps are also 3d printed microcontroller used is raspberry pi pico (RP2040). it has micro usb which i dont like. will use diffrent microcontroller in next version.
